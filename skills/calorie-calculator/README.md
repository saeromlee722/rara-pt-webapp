# Calorie Calculator Skill

개인의 신체 정보를 기반으로 휴식 칼로리(BMR)와 활동 칼로리(TDEE)를 계산하는 스킬입니다.

## 기능

- **BMR (Basal Metabolic Rate)**: Mifflin-St Jeor 공식을 사용한 기초대사량 계산
- **TDEE (Total Daily Energy Expenditure)**: 활동지수를 적용한 총 소비 칼로리 계산
- **추가 분석**: 체지방률, 제지방량 자동 계산

## 입력 형식

CSV 파일에 다음 컬럼이 필요합니다:

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| 고객명 | 이름 | 김철수 |
| 나이 | 만 나이 | 28 |
| 성별 | 남/여 또는 M/F | 남 |
| 키 | 신장(cm) | 175 |
| 몸무게 | 체중(kg) | 70 |
| 골격근량 | 골격근량(kg) | 32 |
| 체지방량 | 체지방량(kg) | 12 |
| 활동지수 | 활동 수준 계수 | 1.55 |

### 활동지수 기준

- **1.2** - 좌식 생활 (운동 거의 안함)
- **1.375** - 가벼운 활동 (주 1-3일 운동)
- **1.55** - 보통 활동 (주 3-5일 운동)
- **1.725** - 활발한 활동 (주 6-7일 운동)
- **1.9** - 매우 활발 (매일 강도높은 운동 또는 육체노동)

## 출력 형식

각 고객별로 `{고객명}.csv` 파일이 생성되며, 다음 정보를 포함합니다:

- 입력된 모든 정보
- **휴식_칼로리_BMR**: 기초대사량 (kcal/day)
- **활동_칼로리_TDEE**: 총 소비 칼로리 (kcal/day)
- **체지방률_%**: 체지방 비율
- **제지방량_kg**: 체중에서 체지방을 제외한 무게

추가로 `summary_all.csv` 파일에 모든 고객의 결과가 통합됩니다.

## 사용법

```bash
python calculate.py sample_input.csv
```

## 예시

### 입력 (sample_input.csv)
```
고객명,나이,성별,키,몸무게,골격근량,체지방량,활동지수
김철수,28,남,175,70,32,12,1.55
```

### 출력 (김철수.csv)
```
고객명,나이,성별,키_cm,몸무게_kg,골격근량_kg,체지방량_kg,활동지수,휴식_칼로리_BMR,활동_칼로리_TDEE,체지방률_%,제지방량_kg
김철수,28,남,175,70,32,12,1.55,1673.75,2594.31,17.14,58.0
```

## BMR 계산 공식 (Mifflin-St Jeor)

- **남성**: BMR = 10 × 체중(kg) + 6.25 × 신장(cm) - 5 × 나이 + 5
- **여성**: BMR = 10 × 체중(kg) + 6.25 × 신장(cm) - 5 × 나이 - 161

## TDEE 계산

TDEE = BMR × 활동지수
