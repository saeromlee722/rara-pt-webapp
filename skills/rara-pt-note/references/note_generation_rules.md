# Note Generation Rules

## Core Philosophy
- 무게보다 정렬
- 개수보다 체감
- 자세 설명보다 힘의 방향
- 문장은 코치 메모형으로 짧게

## Inference Pipeline
1. 운동 패턴 분류
2. 기능적 목적 추론
3. 회원 필요 이유 추론
4. 포인트 3~4개 생성
5. 체감 키워드 생성
6. 보상/체크 신호 생성
7. 다음 수업 방향 추론

## Theme Generation
Create one narrative line in this flow:
- 선활성/준비 → 메인 패턴 → 강화 → 마무리

## Summary Generation
Select 3~4 principles:
- 접지
- 무릎 중심선
- 골반 수평
- 견갑 후인/하강
- 허리 과신전 금지
- 반동 금지
- 무게보다 정렬
- 자극 흐려지면 속도 감소

## Style Constraints
- 짧고 명확하게
- 회원이 바로 이해 가능한 표현
- 해부학 교과서식 문장 금지

Good examples:
- 엉덩이 아래 묵직함
- 겨드랑이 아래 당겨짐
- 날개뼈가 붙는 느낌
- 무릎 중심선 유지
- 골반 수평 유지
