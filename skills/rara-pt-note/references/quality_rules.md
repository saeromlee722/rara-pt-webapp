# 품질 향상 규칙

좋은 수업노트는 운동 하나하나를 따로 설명하는 데서 끝나지 않는다.
반드시 아래를 반영한다.

1. 왜 이 순서인지 암시한다
2. 어떤 패턴을 만들기 위한 조합인지 드러낸다
3. 회원 약점을 운동 조합으로부터 추론한다
4. 다음 수업 방향까지 연결한다

## 금지 사항
- 교과서식 긴 설명
- 운동 이름만 나열
- 모든 운동에 똑같은 포인트 반복
- 회원이 느낄 수 없는 전문용어만 사용
- 특이사항 무시

## 권장 사항
- 감각 언어 사용
- 힘의 방향 설명
- 정렬 기준 반복
- 오늘의 의도와 다음 방향 연결
