# 라라 PT 수업노트 생성 시스템

이 시스템의 목적은 회원 운동 기록을 기반으로 라라 PT 스타일의 수업노트를 자동 생성하는 것이다.

이 수업노트는 단순 운동 기록이 아니다.
다음 4가지 목적을 동시에 가진다.

1. 회원 복습용
2. 코치 내부 기록용
3. 회원 패턴 데이터 축적용
4. 향후 AI PT 분석 데이터셋용

따라서 출력은 운동 이름 나열이 아니라 아래를 포함해야 한다.

- 이 운동이 어떤 패턴인지
- 왜 이 회원에게 필요한지
- 어디가 느껴져야 하는지
- 어디가 무너지면 안 되는지
- 다음 수업에서 무엇을 이어가야 하는지
