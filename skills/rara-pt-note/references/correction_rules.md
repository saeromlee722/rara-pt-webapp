# 특이사항 반영 규칙

특이사항이 입력되면 반드시 전체 해석에 반영한다.

## 대퇴골 안정성 부족 / 내회전 자주 일어남
자동 반영:
- 무릎 중심선 유지 강조
- 중둔근/외전 부족 가능성 추론
- 발 접지, 골반 안정 강조
- 다음 수업 방향에 외전 강화, 싱글레그 안정 포함

## 외전하는 힘 부족
자동 반영:
- 아웃타이, 크램쉘, 밴드 힙쓰러스트 해석 강화
- 스쿼트/레그프레스에서 무릎 바깥 힘 유지 강조
- 중둔근 선활성 필요성 언급

## 광배 활성화 필요
자동 반영:
- 암풀다운, 랫풀다운, 하이로우 연결 강조
- 당기기 순서를 하강 → 후인 → 팔로 설명
- 겨드랑이 아래 체감 언급

## 승모 과개입
자동 반영:
- 견갑 하강
- 하부 승모/전거근 강조
- 목 긴장 없음 표현 추가

## 허리 과개입
자동 반영:
- 복압 유지
- 골반 정렬
- 허리 중립 유지
- 둔근/햄스트링 또는 광배 주도 재강조
