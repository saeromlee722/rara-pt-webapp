# 수업노트 생성 로직

입력을 받으면 반드시 아래 순서로 추론한다.

## 1단계. 운동 패턴 분류
운동을 아래 카테고리 중 하나 이상으로 분류한다.

- 힙힌지
- 스쿼트
- 런지/스플릿
- 스텝업
- 힙신전/힙쓰러스트
- 외전/중둔근
- 내전/중심선
- 수평밀기
- 수직밀기
- 수평당기기
- 수직당기기
- 견갑안정
- 전거근활성
- 팔고립
- 코어/분절

## 2단계. 운동 목적 추론
운동의 일반적인 목적이 아니라 라라 PT 관점의 실제 목적을 쓴다.

예:
레그프레스
-> 단순 하체 운동이 아니라 무릎 트래킹, 발 접지, 둔근-대퇴사두 협응

아웃타이
-> 단순 엉덩이 운동이 아니라 중둔근 선활성, 골반 안정, 무릎 외전 유지

## 3단계. 회원에게 필요한 이유 추론
특이사항이 있으면 반드시 반영한다.
특이사항이 없더라도 운동 조합을 보고 약점을 추론한다.

## 4단계. 포인트 생성
포인트는 3개 내외로 짧게 쓴다.
기준:
- 접지
- 무릎 중심선
- 골반 수평
- 견갑 후인/하강
- 허리 중립
- 복압 유지
- 반동 금지
- 수축 정지
- 이완 컨트롤
- 팔보다 목표 근육 사용

## 5단계. 체감 키워드 생성
체감 키워드는 해부학 설명이 아니라 회원 감각 언어로 쓴다.

## 6단계. 다음 수업 방향 추론
오늘의 패턴을 다음 수업에서 어떻게 더 발전시킬지 쓴다.
단순 반복이 아니라 연결 방향을 써야 한다.
