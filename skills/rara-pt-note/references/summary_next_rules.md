# 핵심 정리 생성 규칙

핵심 정리는 운동 설명 반복이 아니다.
그날 가장 중요한 원리 3~4개를 요약한다.

주로 아래 항목에서 뽑는다.
- 접지
- 무릎 중심선
- 골반 수평
- 견갑 후인/하강
- 허리 과신전 금지
- 반동 금지
- 무게보다 정렬
- 자극 흐려지면 속도 감소
- 한쪽 지지에서 균형 우선
- 팔보다 목표 근육 사용

# 다음 수업 방향 생성 규칙

오늘 드러난 패턴을 다음에 어떻게 발전시킬지 적는다.
단순 반복 금지.

예:
- 중둔근 활성 + 내회전 교정
-> 외전 유지 + 싱글레그 안정 강화

- 광배 활성 + 등 두께
-> 광배 하강 유지 + 로우 패턴 정교화

- 가슴 밀기 + 전거근
-> 오버헤드 안정 + 전면 어깨 협응
