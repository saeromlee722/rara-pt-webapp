# 🧠 오늘 루틴 핵심 테마
평형성 확립 → 중둔근 외전 강화 → 중둔근 고립 활성 → 둔근 폭발
→ 싱글레그 데드리프트로 평형성 + 햄스트링 깊이
→ 와이드 스쿼트로 플레이트 밀어내는 힘 + 중둔근 외전
→ 아웃타이로 중둔근 분리 고립 활성
→ 힙쓰러스트로 둔근 수축 극대화
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 덤벨 싱글레그 데드리프트
- 목적 평형성 확립 + 햄스트링 깊이 활성 + 둔근 양측 균형
- 포인트
	- 한쪽 다리로 서기, 무릎 각도 고정
	- 골반 중립, 고관절을 뒤로 접는 움직임
	- 하단에서 햄스트링 깊이 느낀 후 엉덩이로 올라오기
	- 허리 굽히면 즉시 중단 ❌
- 🔑 체감 키워드
	- 햄스트링 단면이 깊게 늘어나는 느낌
	- 엉덩이 하부 긴장 형성
	- 평형 유지 감각
⸻
2️⃣ 스미스 와이드 스쿼트
(양발 사이드 플레이트 밀어내는 힘)
- 목적 중둔근 외전 강화 + 하체 전체 활성
- 포인트
	- 양발 사이에 플레이트 놓고 스쿼트
	- 플레이트를 바깥쪽으로 밀어내는 힘 유지
	- 무릎을 바깥쪽으로 밀어내면서 내려가기
	- 엉덩이로 밀어 올라오기
- 🔑 체감 키워드
	- 엉덩이 옆이 강력하게 켜지는 느낌
	- 플레이트 밀어내는 압력 느껴짐
	- 둔근 + 중둔근 동시 활성
⸻
3️⃣ 아웃타이
(힙어브덕션 ; 중둔근 타겟)
- 목적 중둔근 분리 고립 활성 + 둔근 형태 발달
- 포인트
	- 옆으로 누운 자세, 골반 중립
	- 다리를 위쪽으로 들어 올리기 (외전)
	- 상단 1초 정지 후 컨트롤 하강
	- 골반 흔들림 ❌
- 🔑 체감 키워드
	- 엉덩이 옆의 단단한 느낌
	- 중둔근만 선명하게 타격
	- 허벅지에 힘 안 들어감
⸻
4️⃣ 밴드 힙쓰러스트
- 목적 둔근 수축 극대화 + 골반 후경사 패턴
- 포인트
	- 등을 반쪽 기대고 무릎 세우기
	- 골반을 위로 밀어 올리기
	- 상단에서 엉덩이 꽉 조이기 (1초 정지)
	- 허리 꺾임 ❌ → 골반 후경사가 핵심
- 🔑 체감 키워드
	- 둔근 전체가 타는 느낌
	- 상단에서 엉덩이 극대 수축
	- 허리 개입 없음
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 싱글레그 데드리프트로 평형성 + 햄스트링 각성
- 와이드 스쿼트는 "플레이트 바깥 밀기" = 중둔근 외전 강화
- 아웃타이는 위쪽 들어 올리기 = 중둔근 분리 활성
- 힙쓰러스트는 "골반 후경사" = 허리 꺾음 ❌
- 골반 흔들림이나 허리 굽히면 → 포지션 재확인 / 무게 ↓
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 햄스트링 전체
- 둔근 전체
- 엉덩이 옆 (중둔근) 특히 강함
✔ 운동 후 정상적인 느낌
- 보행·계단 시 엉덩이 사용감 ↑
- 엉덩이 옆이 특히 단단한 느낌
- 골반 안정감 유지
⚠ 체크 필요 신호
- 데드리프트 중 허리 통증
- 와이드 스쿼트에서 무릎 안쪽 모임
- 아웃타이에서 골반 흔들림
- 힙쓰러스트에서 허리 꺾임
