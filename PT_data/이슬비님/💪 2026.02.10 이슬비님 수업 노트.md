# 🧠 오늘 루틴 핵심 테마
후면 사슬 강화 → 둔근 고립 → 와이드 패턴 안정 → 수축 마무리 → 힙힌지 패턴 선명화 → 엉덩이 단독 수축 인지 → 외전 저항 속 하체 정렬 유지
⸻
# 🏋️‍♀️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 바벨 스티프 데드리프트
- 목적 햄스트링·둔근 후면 사슬 강화
- 포인트
	- 무릎 미세 굴곡 고정
	- 바벨은 정강이 가까이 수직 이동
	- 엉덩이를 뒤로 길게 밀기
	- 허리 중립 유지
- 🔑 체감 키워드
	- 허벅지 뒤가 깊게 늘어나는 느낌
	- 엉덩이 아래 긴장 형성
	- 허리 개입 없음
⸻
2️⃣ 이지바 스티프 데드리프트
- 목적 후면 사슬 자극 밀도 증가 + 반복 패턴 고정
- 포인트
	- 첫 세트보다 더 컨트롤 중심
	- 내려가는 이완 구간 천천히
	- 상체 흔들림 ❌
- 🔑 체감 키워드
	- 햄스트링 자극 선명
	- 엉덩이 아래 피로도 증가
	- 자극 집중도 ↑
⸻
3️⃣ 스탠딩 케이블 킥백
- 목적 둔근 단독 수축 + 골반 안정
- 포인트
	- 허리 흔들림 ❌
	- 다리를 차기보다 엉덩이로 밀어내기
	- 상단 1초 수축 유지
- 🔑 체감 키워드
	- 엉덩이 중앙이 직접 수축
	- 허벅지 개입 최소
	- 골반 안정감 ↑
⸻
4️⃣ 스미스 밴드 와이드 스쿼트
(플레이트 발 바깥쪽에 두고 밀어내는 힘)
- 목적 둔근·내전근 동시 사용 + 외전 저항 강화
- 포인트
	- 발 간격 넓게, 발끝 바깥
	- 발 바깥쪽으로 플레이트 밀어내는 힘 유지
	- 밴드 저항에 밀리지 않게 무릎 바깥 유지
	- 엉덩이 중심 하강
- 🔑 체감 키워드
	- 엉덩이와 허벅지 안쪽 동시 사용
	- 하체 중심이 아래로 단단히 고정
	- 무릎 흔들림 없음
⸻
5️⃣ 힙 쓰러스트 (밴드)
- 목적 둔근 최대 수축 + 골반 안정 마무리
- 포인트
	- 무릎 바깥으로 밀며 밴드 저항 유지
	- 상단에서 골반 말아 올리기
	- 수축 1–2초 유지
- 🔑 체감 키워드
	- 엉덩이 전체가 꽉 차오름
	- 허벅지 힘 빠짐
	- 둔근 펌핑 극대화
⸻
# 📌 오늘 루틴 핵심 정리
- 힌지 → 고립 → 와이드 파워 → 수축 완성 흐름
- 바깥으로 밀어내는 힘이 무너지면 무게 ↓
- 힙쓰러스트는 허리 아닌 엉덩이 수축
- 자극 흐려지면 → 속도 ↓ / 이완 길게 / 범위 정리
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 햄스트링 상부
- 엉덩이 중앙·하부
- 엉덩이 옆 (중둔근)
- 허벅지 안쪽
✔ 운동 후 정상적인 느낌
- 보행 시 엉덩이 사용감 ↑
- 하체가 아래에서 단단히 받쳐주는 느낌
- 무릎 안정감 상승
⚠ 체크 필요 신호
- 데드리프트 중 허리 통증
- 와이드 스쿼트에서 무릎 안쪽 말림
- 힙쓰러스트 시 허리 과신전
- 둔근보다 허벅지 앞 과도한 피로
