# 🧠 오늘 루틴 핵심 테마
고관절 주도 움직임 확립 → 둔근 하부 활성 → 중둔근 외전 강화
→ 스티프 데드리프트로 고관절 주도 패턴 학습
→ 케이블 킥백으로 대둔근 하부 고립 강화
→ 브이스쿼트에서 외전 유지로 중둔근 활성
→ 아웃타이로 대둔근 하부 마무리 강화
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 바벨 스티프 데드리프트
(고관절 움직임 우선)
- 목적 고관절 힙 힌지 패턴 확립 + 햄스트링 깊이 활성
- 포인트
	- 무릎 각도 고정, 고관절을 뒤로 접는 움직임
	- 바벨을 몸 가까이 수직 이동
	- 허리 굳혀지면 즉시 중단 ❌
- 🔑 체감 키워드
	- 허벅지 뒤가 깊게 늘어나는 느낌
	- 엉덩이 아래 긴장 형성
	- 허리 개입 없음
⸻
2️⃣ 케이블 벤치 킥백
- 목적 대둔근 하부 고립 강화 + 둔근 수축 패턴
- 포인트
	- 벤치에 엎드려 코어 고정
	- 다리를 뒤로 차는 동시에 엉덩이 조이기
	- 무릎 세우기 ❌ → 엉덩이 수축이 핵심
- 🔑 체감 키워드
	- 엉덩이 아래쪽이 꽉 조이는 느낌
	- 허벅지 개입 없음
	- 대둔근 하부 선명한 수축
⸻
3️⃣ 밴드 브이스쿼트
(외전하는 힘)
- 목적 하체 힘 + 중둔근 외전 동시 활성화
- 포인트
	- 밴드를 무릎에 걸고 스쿼트 자세
	- 내려갈 때 무릎을 바깥쪽으로 밀어내는 힘 유지
	- 엉덩이로 밀어 올라오면서도 외전 유지
	- 무릎 안쪽으로 모이지 않게 ❌
- 🔑 체감 키워드
	- 둔근 + 엉덩이 옆 동시에 타는 느낌
	- 무릎을 바깥으로 밀어내는 압력 느껴짐
	- 골반 안정감 유지
⸻
4️⃣ 아웃타이 (대둔근 하부 타겟)
- 목적 대둔근 하부 고립 강화 + 둔근 형태 발달
- 포인트
	- 옆으로 누운 자세, 골반 중립
	- 무릎을 뒤쪽 대각선 방향으로 들어 올리기
	- 상단 1초 정지 후 컨트롤 하강
- 🔑 체감 키워드
	- 엉덩이 아래쪽의 단단한 느낌
	- 대둔근 하부만 선명하게 타격
	- 허벅지에 힘 안 들어감
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 스티프 데드리프트는 고관절을 뒤로 접는 움직임 우선
- 킥백은 "다리 차기 + 엉덩이 조이기" 동시에
- 브이스쿼트는 "무릎 바깥으로 밀기" = 중둔근 활성
- 아웃타이는 뒤쪽 대각선으로 = 대둔근 하부 타겟
- 허리 개입되거나 무릎 모이면 → 동작 재확인 / 무게 ↓
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 햄스트링 전체
- 엉덩이 아래쪽 (대둔근 하부)
- 엉덩이 옆 (중둔근)
✔ 운동 후 정상적인 느낌
- 보행·계단 시 엉덩이 사용감 ↑
- 하체 뒤쪽이 길고 단단해진 느낌
- 엉덩이 아래쪽이 특히 단단함
⚠ 체크 필요 신호
- 데드리프트 중 허리 통증
- 킥백에서 허벅지 과도한 개입
- 브이스쿼트에서 무릎 모이는 느낌
- 둔근보다 허벅지 앞 과도한 피로
