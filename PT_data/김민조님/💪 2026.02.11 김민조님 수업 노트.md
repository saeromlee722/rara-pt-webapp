# 🧠 오늘 루틴 핵심 테마
후면 어깨 안정 → 견갑 정렬 → 오버헤드 밀기 패턴 → 측면 라인 완성 → 어깨 전면 과개입 방지 → 후면·측면 삼각근 균형 강화 → 승모 개입 최소화
⸻
# 🏋️‍♀️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 비하인드 펙덱 플라이 머신
- 목적 후면 삼각근 활성 + 견갑 후인 안정
- 포인트
	- 가슴 패드 밀착
	- 팔로 벌리기 ❌ → 어깨 뒤로 열기
	- 반동 없이 끝 지점 1초 정지
- 🔑 체감 키워드
	- 어깨 뒤쪽이 또렷하게 수축
	- 등 중앙이 안정됨
	- 목 긴장 없음
⸻
2️⃣ 케이블 로프 페이스풀
- 목적 후면 삼각근 + 중부 승모 협응
- 포인트
	- 당기기 전 견갑 후인
	- 로프를 눈높이 방향으로 벌리며 당김
	- 팔꿈치 높이 유지
- 🔑 체감 키워드
	- 어깨 뒤·등 위가 동시에 조여짐
	- 승모 과긴장 없음
	- 어깨 위치가 낮아짐
⸻
3️⃣ 비하인드 넥프레스
- 목적 측면 삼각근 중심 오버헤드 밀기
- 포인트
	- 허리 과신전 ❌
	- 팔꿈치가 바깥 방향 유지
	- 귀 옵 수직 궤적 유지
- 🔑 체감 키워드
	- 어깨 옵 라인이 직접적으로 타는 느낌
	- 가슴 개입 최소
	- 목 긴장 없음
⸻
4️⃣ 덤벨 와이 프레스
- 목적 전·측면 삼각근 협응 + 안정적 밀기
- 포인트
	- 덤벨이 Y자 궤적으로 상승
	- 복압 유지, 갈비 들림 ❌
	- 상단에서 컨트롤 유지
- 🔑 체감 키워드
	- 어깨 전·측면 동시 사용
	- 흔들림 없는 밀기
	- 상단 수축 안정
⸻
5️⃣ 사이드 레터럴 레이즈 머신
- 목적 측면 삼각근 라인 집중 강화
- 포인트
	- 반동 ❌
	- 팔꿈치가 손보다 약간 위
	- 상단 1초 정지
	- 승모 힘 빼기
- 🔑 체감 키워드
	- 어깨 옵 라인이 타는 느낌
	- 좌우 대칭 유지
	- 상체 고정 안정
⸻
# 📌 오늘 루틴 핵심 정리
- 후면 활성 → 오버헤드 밀기 → 측면 라인 완성
- 오버헤드 동작은 허리 아닌 어깨 주도
- 측면 운동은 무게보다 정렬
- 자극 흐려지면 → 무게 ↓ / 속도 ↓ / 정지 ↑
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 어깨 후면
- 어깨 측면
- 등 위 중앙 (중부 승모)
✔ 운동 후 정상적인 느낌
- 어깨 라인이 또렷해짐
- 어깨 위치가 안정됨
- 목·승모 부담 없음
⚠ 체크 필요 신호
- 넥프레스 중 허리 과신전
- 레터럴 레이즈에서 반동 사용
- 페이스풀 시 목 긴장
- 어깨 앞 통증 발생
