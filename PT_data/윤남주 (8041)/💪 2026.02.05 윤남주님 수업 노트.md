# 🧠 오늘 루틴 핵심 테마
평형성 확립 → 내전근 활성 → 중둔근 외전 → 둔근 폭발 → 정렬 패턴 강화
→ 싱글레그 데드리프트로 평형성 + 햄스트링 각성
→ 이너레그업으로 내전근 분리 활성
→ 밴드 브이스쿼트로 중둔근 외전 강화
→ 힙쓰러스트로 둔근 수축 극대화
→ 레그프레스로 발바닥-발목-무릎-골반 정렬 패턴 확립
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 덤벨 싱글레그 스티프 데드리프트
- 목적 평형성 확립 + 햄스트링 깊이 활성 + 둔근 양측 균형
- 포인트
	- 한쪽 다리로 서기, 무릎 각도 고정
	- 골반 중립, 고관절을 뒤로 접는 움직임
	- 하단에서 햄스트링 깊이 느낀 후 엉덩이로 올라오기
	- 허리 굳히면 즉시 중단 ❌
- 🔑 체감 키워드
	- 햄스트링 단면이 깊게 늘어나는 느낌
	- 엉덩이 하부 긴장 형성
	- 평형 유지 감각
⸻
2️⃣ 이너레그업
- 목적 내전근 분리 활성 + 골반저 안정
- 포인트
	- 옆으로 누워 한쪽 다리 들어 올리기
	- 골반 고정, 다리를 안쪽으로 들어 올리는 느낌
	- 무릎이 안쪽으로 향하게
	- 상단 1초 정지
- 🔑 체감 키워드
	- 허벅지 안쪽이 조이는 느낌
	- 내전근만 선명하게 켜짐
	- 골반 흔들림 없음
⸻
3️⃣ 밴드 브이스쿼트
(외전하는 힘)
- 목적 하체 힘 + 중둔근 외전 동시 활성화
- 포인트
	- 밴드를 무릎에 걸고 스쿼트 자세
	- 내려갈 때 무릎을 바깥쪽으로 밀어내는 힘 유지
	- 엉덩이로 밀어 올라오면서도 외전 유지
	- 무릎 안쪽으로 모이지 않게 ❌
- 🔑 체감 키워드
	- 둔근 + 엉덩이 옆 동시에 타는 느낌
	- 무릎을 바깥으로 밀어내는 압력 느껴짐
	- 골반 안정감 유지
⸻
4️⃣ 스미스머신 힙쓰러스트
- 목적 둔근 수축 극대화 + 골반 후경사 패턴
- 포인트
	- 등을 반쪽 기대고 무릎 세우기
	- 골반을 위로 밀어 올리기
	- 상단에서 엉덩이 꽉 조이기 (1초 정지)
	- 허리 꽂임 ❌ → 골반 후경사가 핵심
- 🔑 체감 키워드
	- 둔근 전체가 타는 느낌
	- 상단에서 엉덩이 극대 수축
	- 허리 개입 없음
⸻
5️⃣ 레그프레스
(밴드 발목 ; 발바닥 - 발목-무릎-골반 정렬)
- 목적 하체 정렬 패턴 확립 + 하체 전체 활성
- 포인트
	- 발목에 밴드 걸기 (외전 유지 목적)
	- 발바닥 중앙 → 발목 → 무릎 → 골반 수직 정렬 확인
	- 발목이 안쪽으로 모이지 않게 ❌
	- 엉덩이로 밀어 올리기
- 🔑 체감 키워드
	- 하체 전체가 정렬된 느낌
	- 발바닥부터 힘이 전달되는 감각
	- 골반 안정적
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 싱글레그 데드리프트로 평형성 + 햄스트링 각성
- 이너레그업으로 내전근 분리 활성
- 브이스쿼트는 "무릎 바깥으로 밀기" = 중둔근 활성
- 힙쓰러스트는 "골반 후경사" = 허리 꽂음 ❌
- 레그프레스는 발바닥-발목-무릎-골반 수직 정렬이 핵심
- 골반 흔들림이나 허리 굳히면 → 포지션 재확인 / 무게 ↓
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 햄스트링 전체
- 둔근 전체
- 엉덩이 옆 (중둔근)
- 허벅지 안쪽 (내전근)
✔ 운동 후 정상적인 느낌
- 보행·계단 시 엉덩이 사용감 ↑
- 하체 전체가 정렬된 느낌
- 내전근 + 중둔근 동시 사용감
⚠ 체크 필요 신호
- 데드리프트 중 허리 통증
- 이너레그업에서 골반 흔들림
- 브이스쿼트에서 무릎 모이는 느낌
- 레그프레스에서 발목 안쪽 모임
