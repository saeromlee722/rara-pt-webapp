# 🧠 오늘 루틴 핵심 테마
견갑 후인 안정 → 등 중앙 수축 → 견갑 밀어내기 + 승모 활성 → 광배 하강
→ 시티드 로우로 후인 잡아두는 힘 확립
→ 암풀다운으로 광배 중앙 분리 수축
→ 숄더 프레스에서 리버스 쉬러그로 중부승모 활성
→ 랫풀다운으로 견갑 하강 → 광배 전체 확보
→ 플레이트 로디드 로우로 등 전체 마무리
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 시티드 케이블 로우
(후인 잡아두는 힘)
- 목적 견갑 후인 안정 + 등 중앙 수축 패턴 확립
- 포인트
	- 당기기 전 견갑 후인 먼저 잡기
	- 후인 위치를 끝까지 유지하며 팔 당기기
	- 팔로 당기기 ❌ → 견갑 주도 당기기
- 🔑 체감 키워드
	- 견갑이 등 중앙으로 모이는 느낌
	- 후인 위치 끝까지 고정
	- 등 중앙 수축 선명
⸻
2️⃣ 케이블 암풀다운
- 목적 광배 중앙 분리 사용 + 팔꿈치 통제
- 포인트
	- 팔꿈치 살짝 굽힌 상태 유지
	- 팔을 옆구리쪽으로 당기는 느낌
	- 어깨 올라가지 않게 ❌
- 🔑 체감 키워드
	- 광배 중앙이 조이는 느낌
	- 팔 위치 제어감
	- 어깨 안정적
⸻
3️⃣ 스미스 숄더 프레스
(+리버스 쉬러그)
- 목적 중부승모 활성 + 견갑 밀어내기 + 어깨 프레스
- 포인트
	- 시작 전 리버스 쉬러그로 중부승모 활성화
	- 견갑을 위로 밀어내듯이 프레스
	- 정수리 방향 수직 궤적
- 🔑 체감 키워드
	- 승모 중간부터 켜지는 느낌
	- 견갑이 위로 밀리는 감각
	- 어깨 앞 힘 최소화
⸻
4️⃣ 랫풀다운
- 목적 견갑 하강 + 광배 전체 활성
- 포인트
	- 견갑 하강부터 시작
	- 팔꿈치는 어깨 바로 아래로
	- 가슴까지 당기기, 어깨 올라가지 않게 ❌
- 🔑 체감 키워드
	- 광배 전체가 당겨지는 느낌
	- 견갑 아래로 단단히 고정
	- 가슴 위쪽 수축감
⸻
5️⃣ 플레이트 로디드 시티드 로우
- 목적 등 전체 수축 + 견갑 후인 강화
- 포인트
	- 견갑 후인 먼저, 그 다음 팔 당기기
	- 상체 반동 ❌
	- 견갑 모으는 위치까지 완전 후인
- 🔑 체감 키워드
	- 등 중앙·광배 전체 수축
	- 견갑 후인 강력하게 느껴짐
	- 상체 안정적
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 모든 로우 동작은 견갑 후인 먼저 → 팔 당김
- 숄더 프레스는 리버스 쉬러그로 승모 활성 후 견갑 밀기
- 풀다운은 견갑 하강이 핵심
- 어깨 올라가면 → 견갑 세팅 재확인 / 무게 ↓
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 등 중앙
- 광배 중앙·하부
- 승모 중간부
✔ 운동 후 정상적인 느낌
- 등과 어깨가 동시에 단단한 느낌
- 견갑이 안정된 사용감
- 목·승모 부담 없음
⚠ 체크 필요 신호
- 로우 동작에서 어깨 들림
- 풀다운에서 팔만 사용되는 느낌
- 숄더 프레스 후 목 과긴장
- 로우 중 상체 과도한 반동
