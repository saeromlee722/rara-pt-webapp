# 🧠 오늘 루틴 핵심 테마
기능적 둔근 활성 → 햄스트링 깊이 각성 → 중둔근 외전 강화
→ 박스 스텝업으로 둔근 기능 + 단단함 확보
→ 스티프 데드리프트(앞발 높게)로 햄스트링 가동범위 극대화
→ 브이스쿼트에서 외전하는 힘 유지로 중둔근 활성
→ 아웃타이로 중둔근 고립 강화 + 둔근 외측 라인 형성
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 박스 스텝업
- 목적 둔근 기능적 활성 + 단단함 + 단측 안정성
- 포인트
	- 박스에 올라갈 때 엉덩이로 밀어 올리기
	- 무릎 안쪽으로 급격히 꼬이지 않게 ❌
	- 상단에서 엉덩이 꽉 조이기
- 🔑 체감 키워드
	- 엉덩이가 힘있게 밀어 올리는 느낌
	- 골반 안정감
	- 허벅지 앞 개입 최소
⸻
2️⃣ 덤벨 스티프 데드리프트
(앞발 높게)
- 목적 햄스트링 스트레치 극대화 + 둔근 후면 활성
- 포인트
	- 앞발을 살쩍 높여 가동 범위 확보
	- 무릎 각도 고정
	- 덤벨을 몸 가까이 수직 이동
	- 엉덩이를 뒤로 길게 밀기
- 🔑 체감 키워드
	- 허벅지 뒤가 깊게 늘어나는 느낌
	- 엉덩이 아래 긴장 형성
	- 허리 개입 없음
⸻
3️⃣ 밴드 브이스쿼트머신
(외전하는 힘)
- 목적 하체 힘 + 중둔근 외전 동시 활성화
- 포인트
	- 밴드를 무릎에 걸고 스쿼트 자세
	- 내려갈 때 무릎을 바깥쪽으로 밀어내는 힘 유지
	- 엉덩이로 밀어 올라오면서도 외전 유지
	- 무릎 안쪽으로 모이지 않게 ❌
- 🔑 체감 키워드
	- 둔근 + 엉덩이 옆 동시에 타는 느낌
	- 무릎을 바깥으로 밀어내는 압력 느껴짐
	- 골반 안정감 유지
⸻
4️⃣ 아웃타이 (힙 어브덕션 ; 중둔근 타겟)
- 목적 중둔근 고립 강화 + 둔근 형태 발달
- 포인트
	- 옆으로 누운 자세, 골반 중립
	- 무릎을 칼로 쑤듯이 위로 가져가기
	- 상단 1초 정지 후 컨트롤 하강
- 🔑 체감 키워드
	- 엉덩이 옆의 블록 모양이 느껴짐
	- 중둔근만 선명하게 타격
	- 허벅지 외측에 힘 안 들어감
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 박스 스텝업으로 둔근 기능 + 단측 안정성 확보
- 스티프 데드리프트(앞발 높게)는 햄스트링 깊이 각성
- 브이스쿼트는 "무릎 바깥으로 밀기" = 중둔근 활성
- 아웃타이로 중둔근 고립 강화 + 엉덩이 옆 라인
- 무릎 안쪽으로 모이면 → 외전 힘 재확인 / 골반 위치 체크
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 햄스트링 상부
- 엉덩이 전체 (특히 하부)
- 엉덩이 옆 (중둔근)
✔ 운동 후 정상적인 느낌
- 보행·계단 시 엉덩이 사용감 ↑
- 하체 뒤쪽이 길고 단단해진 느낌
- 엉덩이 옆 라인이 느껴짐
⚠ 체크 필요 신호
- 데드리프트 중 허리 통증
- 스텝업 시 무릎 급격히 안쪽 꼬임
- 브이스쿼트에서 무릎 모이는 느낌
- 둔근보다 허벅지 앞 과도한 피로
