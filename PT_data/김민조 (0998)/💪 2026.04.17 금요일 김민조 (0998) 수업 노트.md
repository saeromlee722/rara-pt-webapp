### 1️⃣ 스미스 굿모닝 (햄스트링)

_(힌지 패턴)_

- **역할**
 스미스 · 햄스트링 기준으로 자극 위치를 분리해서 확인

- **목적**
 후면 체인 텐션 형성 + 골반 힌지 감각 강화 + 무릎 각도 유지

- **포인트**

 - 엉덩이를 뒤로 접어 햄스트링을 길게 만든다

 - 무릎 각도는 거의 고정

 - 허리 중립 유지

 - 올라올 때 엉덩이로 바닥을 민다

- **🔑 체감 키워드**

 - 허벅지 뒤 길어짐

 - 엉덩이 아래 묵직함

 - 허리 편안함

- **흔한 오류**

 - 허리 말림

 - 무릎 과굴곡

 - 반동으로 올라오기

- **다음 연결**
 햄스트링 텐션 유지 후 둔근 수축 운동으로 연결

### 2️⃣ 펜듈럼 스쿼트 머신 (하체 전체)

_(squat)_

- **역할**
 main exercise

- **목적**
 하체 스쿼트 패턴 강화 + 대퇴사두 + 둔근 협응

- **포인트**

 - 발바닥 전체 접지

 - 무릎-발 방향 유지

 - 하단 깊이 안정

 - 올라올 때 둔근 사용

- **🔑 체감 키워드**

 - 허벅지 앞 단단함

 - 엉덩이 같이 사용

 - 무릎 안정감

- **흔한 오류**

 - 무릎 안쪽 붕괴

 - 발 아치 붕괴

 - 허리 과신전

- **다음 연결**
 힙힌지 + 힙쓰러스트 연결

### 3️⃣ 킥백 머신 (대둔근)

_(신전 패턴)_

- **역할**
 머신 · 대둔근 기준으로 자극 위치를 분리해서 확인

- **목적**
 고관절 신전 감각 강화 + 둔근 후면 수축 선명화 + 허리 보상 감소

- **포인트**

 - 골반을 살짝 말아 둔근으로 잠근다

 - 허리 꺾어서 올리지 않기

 - 발바닥 접지 유지

 - 끝지점에서 둔근 수축을 먼저 확인

- **🔑 체감 키워드**

 - 엉덩이 뒤쪽 묵직함

 - 상단 둔근 잠김

 - 허리 부담 없음

- **흔한 오류**

 - 허리 과신전

 - 햄스트링 쥐남

 - 골반이 먼저 풀림

- **다음 연결**
 대둔근 수축 확인 후 힌지/스쿼트 메인 패턴으로 연결


---

### 4️⃣ 와이드 스쿼트 머신 (하체 전체)

_(정렬 패턴)_

- **역할**
 머신 · 하체 전체 기준으로 자극 위치를 분리해서 확인

- **목적**
 하체 전체 협응 + 발-무릎-골반 정렬 안정 + 둔근과 대퇴사두의 힘 분배

- **포인트**

 - 발바닥 전체를 균등하게 누른다

 - 무릎과 발끝 방향을 맞춘다

 - 골반이 한쪽으로 빠지지 않게 유지

 - 밀어낼 때 허리보다 하체 전체 압력으로 올라온다

- **🔑 체감 키워드**

 - 하체 전체 단단함

 - 발바닥 접지감

 - 무릎 흔들림 감소

- **흔한 오류**

 - 무릎 안쪽 붕괴

 - 발 접지 풀림

 - 허리로 버티기

- **다음 연결**
 하체 전체 정렬 확인 후 스쿼트/런지/힌지 패턴으로 연결
