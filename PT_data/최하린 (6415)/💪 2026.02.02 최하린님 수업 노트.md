# 🧠 오늘 루틴 핵심 테마
햄스트링 선활성 → 중둔근 각성 → 기능적 런지 패턴 → 둔근 외전 강화
→ 굿모닝으로 햄스트링 길이 감각 확보
→ 크램쉘로 골반 안정 + 중둔근 선활성화
→ 싱글 데드리프트로 좌우 균형 확립
→ 크로스 런지로 둔근 깊이 + 안정성 강화
→ 아웃타이(힙 어브덕션)로 중둔근 외전 극대화
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 굿모닝 엑서사이즈
- 목적 햄스트링 스트레치 강화 + 둔근 후면 선활성
- 포인트
	- 무릎 각도 고정, 엉덩이 뒤로 길게 밀기
	- 허리 굳혀지면 즉시 중단 ❌
	- 상체를 기울일 때 햄스트링 늘어나는 감각 유지
- 🔑 체감 키워드
	- 허벅지 뒤가 깊게 늘어나는 느낌
	- 엉덩이 아래 긴장 형성
	- 허리 개입 없음
⸻
2️⃣ 덤벨 싱글 스티프 데드리프트
- 목적 한쪽 햄스트링 + 둔근 집중 강화 + 좌우 균형
- 포인트
	- 작용쪽 다리는 무릎 고정, 반대쪽은 가볍게 지탱
	- 덤벨을 작용쪽 다리 가까이 수직 이동
	- 골반 돌아가지 않게 ❌
- 🔑 체감 키워드
	- 한쪽 햄스트링이 깊이 늘어나는 느낌
	- 작용쪽 둔근 하부 선명하게 수축
	- 골반 안정감 유지
⸻
3️⃣ 밴드 크램쉘
- 목적 중둔근 선활성 + 골반 안정 준비
- 포인트
	- 골반 뒤로 말리지 않게 고정
	- 발은 붙인 채 무릎만 열기
	- 상단에서 1초 정지
- 🔑 체감 키워드
	- 엉덩이 옆이 먼저 켜지는 느낌
	- 골반이 단단히 잡힘
	- 허벅지 개입 최소
⸻
4️⃣ 스미스 크로스 런지
- 목적 둔근 깊이 활성 + 중둔근 동시 사용 + 골반 안정성
- 포인트
	- 뒤쪽 무릎을 깊게 굽혀 내려가기
	- 앞무릎 90도, 골반 흔들림 ❌
	- 일어서면서 엉덩이로 밀기
- 🔑 체감 키워드
	- 둔근이 깊이 늘어나다 수축되는 느낌
	- 엉덩이 옆 + 엉덩이 하부 동시 사용
	- 무릎 흔들림 없이 안정적
⸻
5️⃣ 스탠딩 밴드 아웃타이 (힙 어브덕션)
- 목적 중둔근 외전 기능 강화 + 골반 안정 패턴
- 포인트
	- 선 자세, 코어 단단히
	- 다리를 옆으로 밀듯이 외전
	- 골반 회전 ❌, 상체 기울어짐 ❌
- 🔑 체감 키워드
	- 엉덩이 옆이 타는 느낌
	- 하체 전체 안정감
	- 허벅지 힘 빠짐
⸻
6️⃣ 아웃타이 (힙 어브덕션 ; 중둔근 타겟)
- 목적 중둔근 고립 강화 + 둔근 형태 발달
- 포인트
	- 옆으로 누운 자세, 골반 중립
	- 무릎을 칼로 쑤듯이 위로 가져가기
	- 상단 1초 정지 후 컨트롤 하강
- 🔑 체감 키워드
	- 엉덩이 옆의 블록 모양이 느껴짐
	- 중둔근만 선명하게 타격
	- 허벅지 외측에 힘 안 들어감
⸻

📌 오늘 루틴 핵심 정리
- 굿모닝으로 햄스트링 각성 → 크램쉘로 골반 안정
- 싱글 데드리프트로 좌우 균형 + 둔근 하부 집중
- 크로스 런지는 "깊이 굽힌 후 엉덩이로 밀기"
- 아웃타이로 중둔근 외전 기능 극대화
- 골반 돌아가거나 흔들리면 → 무게 ↓ / 코어 재세팅
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 햄스트링 상부
- 엉덩이 아래쪽 (대둔근 하부)
- 엉덩이 옆 (중둔근)
✔ 운동 후 정상적인 느낌
- 보행·계단 시 엉덩이 사용감 ↑
- 하체 뒤쪽이 길고 단단해진 느낌
- 골반 안정감 향상
⚠ 체크 필요 신호
- 데드리프트 중 허리 통증
- 런지 시 무릎 흔들림
- 크램쉘에서 골반 회전 느낌
- 둔근보다 허벅지 앞 과도한 피로
