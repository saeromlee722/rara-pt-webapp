# 🧠 오늘 루틴 핵심 테마
가슴 선활성 → 가슴 주도 프레스 → 어깨 전면 분리 활성 → 광배 하강
→ 펙덱 플라이로 가슴 이완·수축 선활성
→ 스미스 벤치 프레스로 가슴 주도 밀기 패턴
→ 프론트 레이즈로 덤벨 붙이는 힘 + 내전근 활성
→ 프론트 프레스로 팔꿈치 모으는 힘 + 전면 삼각근
→ 맥그립 랫풀다운으로 광배 하강 + 전체 활성
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 펙덱 플라이 머신
- 목적 가슴 이완·수축 인지 + 프레스 전 가슴 선활성
- 포인트
	- 견갑 고정, 팔꿈치 각도 유지
	- 가슴이 열리는 범위까지만
	- 반동 ❌, 하단 컨트롤 필수
- 🔑 체감 키워드
	- 가슴이 넓게 늘어나는 느낌
	- 어깨 개입 없음
	- 가슴 중앙 수축 선명
⸻
2️⃣ 스미스 벤치 프레스
- 목적 가슴 주도 안정적 밀기 패턴 확립
- 포인트
	- 시작 전 견갑 후인·하강 세팅
	- 팔로 밀기 ❌ → 가슴으로 받쳐 올리기
	- 궤적 일정, 상체 흔들림 ❌
- 🔑 체감 키워드
	- 가슴 전체가 차오르는 느낌
	- 어깨 앞 힘 빠짐
	- 프레스 중 안정감 ↑
⸻
3️⃣ 덤벨 프론트 레이즈
(두 개 붙이는 힘)
- 목적 전면 삼각근 + 가슴 내전근 동시 활성
- 포인트
	- 덤벨 두 개를 서로 붙이는 힘 유지하며 들어 올리기
	- 팔꿈치 살짝 굽힌 상태 고정
	- 어깨 높이까지만 → 승모 개입 방지 ❌
	- 내려올 때도 덤벨 붙이는 힘 유지
- 🔑 체감 키워드
	- 가슴 안쪽 + 전면 어깨 동시 수축
	- 덤벨 붙이는 압력 느껴짐
	- 승모 긴장 없이 어깨만 작동
⸻
4️⃣ 덤벨 프론트 프레스
(덤벨 하나로 들고 팔꿈치 모아주는 힘)
- 목적 전면 삼각근 분리 + 팔꿈치 모으는 힘 강화
- 포인트
	- 덤벨 하나를 양손으로 잡고 수직 프레스
	- 팔꿈치를 서로 모아주는 힘 유지
	- 정수리 방향 수직 궤적
- 🔑 체감 키워드
	- 전면 삼각근만 선명하게 켜짐
	- 팔꿈치 모으는 압력 느껴짐
	- 목·승모 힘 빠짐
⸻
5️⃣ 맥그립 랫풀다운
- 목적 광배 하강 + 광배 전체 활성
- 포인트
	- 손바닥 마주보게 그립
	- 견갑 하강부터 시작
	- 가슴까지 당기기, 어깨 올라가지 않게 ❌
- 🔑 체감 키워드
	- 광배 전체가 당겨지는 느낌
	- 견갑 아래로 단단히 고정
	- 가슴 위쪽 수축감
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 플라이로 가슴 먼저 각성 → 프레스 계열
- 벤치 프레스는 "받쳐 올리기", 팔로 밀기 ❌
- 프론트 레이즈는 덤벨 붙이는 힘으로 내전근 활성
- 프론트 프레스는 팔꿈치 모으는 힘이 핵심
- 어깨 앞 긴장 느껴지면 → 무게 ↓ / 견갑 세팅 재확인
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 가슴 전체 (상부·중앙)
- 전면 삼각근 (어깨 앞)
- 가슴 안쪽 (내전근)
- 광배 전체
✔ 운동 후 정상적인 느낌
- 가슴이 촉촉하게 차오른 느낌
- 어깨가 안정된 사용감
- 가슴 안쪽부터 펌핑감
⚠ 체크 필요 신호
- 프레스 후 어깨 앞 날카로운 통증
- 목·승모 과도한 긴장
- 프론트 레이즈 중 어깨 들림 느낌
- 가슴보다 팔에만 피로 집중
