# 🧠 오늘 루틴 핵심 테마
후면 사슬 강화 → 둔근 고립 → 하체 파워 → 팔 분리 마무리 → 힙힌지 패턴 선명화 → 둔근 단독 수축 인지 → 상·하체 분리 사용 능력 강화
⸻
# 🏋️‍♀️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 바벨 스티프 데드리프트
- 목적 햄스트링·둔근 후면 사슬 강화
- 포인트
	- 무릎 미세 굴곡 고정
	- 바벨은 정강이 가까이 수직 이동
	- 엉덩이를 뒤로 길게 밀기
	- 허리 중립 유지
- 🔑 체감 키워드
	- 허벅지 뒤가 길게 늘어나는 느낌
	- 엉덩이 아래 긴장 증가
	- 허리 개입 없음
⸻
2️⃣ 케이블 킥백
(테이블탑 자세)
- 목적 둔근 고립 수축 + 골반 안정
- 포인트
	- 테이블탑 자세에서 허리 고정
	- 다리를 뒤로 "차기"보다 엉덩이로 밀어내기
	- 상단 1초 수축 유지
- 🔑 체감 키워드
	- 엉덩이 중앙이 직접적으로 수축
	- 허벅지 개입 최소
	- 허리 흔들림 없음
⸻
3️⃣ 밴드 브이스쿼트 머신
- 목적 둔근·대퇴사두 동시 사용 + 무릎 정렬 강화
- 포인트
	- 밴드 저항에 밀리지 않게 무릎 바깥 유지
	- 내려갈수록 엉덩이 중심
	- 발바닥 전체 밀착
- 🔑 체감 키워드
	- 엉덩이–허벅지 동시 사용
	- 무릎 흔들림 없음
	- 하체 중심 안정감 ↑
⸻
4️⃣ 케이블 이두 컬
- 목적 이두 단독 수축 인지
- 포인트
	- 팔꿈치 고정
	- 상체 흔들림 ❌
	- 천천히 이완, 상단 1초 정지
- 🔑 체감 키워드
	- 팔 앞쪽이 또렷하게 수축
	- 어깨 개입 없음
	- 좌우 균형 유지
⸻
5️⃣ 케이블 로프 풀다운 (삼두)
- 목적 삼두 단독 신전 + 팔 후면 강화
- 포인트
	- 팔꿈치 몸통 옆 고정
	- 끝까지 완전 신전
	- 반동 ❌, 천천히 이완
- 🔑 체감 키워드
	- 팔 뒤쪽이 타는 느낌
	- 어깨 움직임 최소
	- 상완 고정 안정
⸻
# 📌 오늘 루틴 핵심 정리
- 후면 사슬 → 둔근 고립 → 하체 파워 → 팔 분리 흐름
- 힙힌지는 허리 아닌 엉덩이 주도
- 팔 운동은 "몸 흔들기" 금지
- 자극 흐려지면 → 무게 ↓ / 속도 ↓ / 이완 길게
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 햄스트링 상부
- 엉덩이 중앙
- 허벅지 앞
- 이두·삼두
✔ 운동 후 정상적인 느낌
- 하체 뒤쪽이 길고 단단한 느낌
- 계단 오를 때 엉덩이 사용감 ↑
- 팔이 단단하게 차오른 느낌
⚠ 체크 필요 신호
- 데드리프트 중 허리 통증
- 킥백에서 허리 흔들림
- 브이스쿼트 시 무릎 안쪽 말림
- 팔 운동 중 어깨 개입 과도
