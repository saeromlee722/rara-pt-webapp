# 🧠 오늘 루틴 핵심 테마
중둔근 선활성 → 기능적 런지 패턴 → 흉추 신전 유지 + 하체 깊이 → 둔근 폭발
→ 크램쉘로 골반 안정 + 중둔근 각성
→ 크로스 런지로 둔근 깊이 + 중둔근 동시 사용
→ 와이드 스쿼트에서 흉추 세우기 유지 + 둔근 깊이
→ 힙쓰러스트로 둔근 수축 극대화
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 밴드 크램쉘
- 목적 중둔근 선활성 + 골반 안정 준비
- 포인트
	- 골반 뒤로 말리지 않게 고정
	- 발은 붙인 채 무릎만 열기
	- 상단에서 1초 정지
- 🔑 체감 키워드
	- 엉덩이 옆이 먼저 켜지는 느낌
	- 골반이 단단히 잡힘
	- 허벅지 개입 최소
⸻
2️⃣ 스미스 크로스 런지
- 목적 둔근 깊이 활성 + 중둔근 동시 사용 + 골반 안정성
- 포인트
	- 뒤쪽 무릎을 깊게 굽혀 내려가기
	- 앞무릎 90도, 골반 흔들림 ❌
	- 일어서면서 엉덩이로 밀기
- 🔑 체감 키워드
	- 둔근이 깊이 늘어나다 수축되는 느낌
	- 엉덩이 옆 + 엉덩이 하부 동시 사용
	- 무릎 흔들림 없이 안정적
⸻
3️⃣ 덤벨 와이드 스쿼트
(흉추 신전 상태 유지)
- 목적 둔근 깊이 + 흉추 세우기 패턴 + 하체 전체 활성
- 포인트
	- 가슴 펴고 흉추 세운 상태 유지
	- 발 넓게, 발끝 살짝 바깥
	- 내려갈 때 흉추 굴곡 ❌ → 가슴 펴 상태 유지
	- 엉덩이로 밀어 올라오기
- 🔑 체감 키워드
	- 둔근이 깊이 늘어나는 느낌
	- 가슴이 펴진 상태 유지
	- 하체 전체 단단함
⸻
4️⃣ 밴드 힙쓰러스트
- 목적 둔근 수축 극대화 + 골반 전방 후방 경사 패턴
- 포인트
	- 등을 반쪽 기대고 무릎 세우기
	- 골반을 위로 밀어 올리기
	- 상단에서 엉덩이 꽉 조이기 (1초 정지)
	- 허리 ❌ → 골반 전방 후방 경사가 핵심
- 🔑 체감 키워드
	- 둔근 전체가 타는 느낌
	- 상단에서 엉덩이 극대 수축
	- 허리 개입 없음
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 크램쉘로 골반 먼저 안정 → 중둔근 각성
- 크로스 런지는 "깊이 굽힌 후 엉덩이로 밀기"
- 와이드 스쿼트는 흉추 세우기 = 가슴 펴 상태 유지
- 힙쓰러스트는 "골반 후경사" = 허리  ❌
- 흉추 굴곡되거나 허리에 꽂히면 → 포지션 재확인 / 무게 ↓
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 둔근 전체 (특히 하부)
- 엉덩이 옆 (중둔근)
- 허벅지 뒤
✔ 운동 후 정상적인 느낌
- 보행·계단 시 엉덩이 사용감 ↑
- 하체 전체가 단단한 느낌
- 가슴이 펴진 자세 유지
⚠ 체크 필요 신호
- 런지 시 무릎 흔들림
- 스쿼트에서 흔추 굴곡 느낌
- 힙쓰러스트에서 허리 꽂임
- 둔근보다 허벅지 앞 과도한 피로
