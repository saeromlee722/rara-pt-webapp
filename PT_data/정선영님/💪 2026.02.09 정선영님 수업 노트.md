# 🧠 오늘 루틴 핵심 테마
견갑 후인 안정 → 등 두께 강화 → 광배 하강 → 어깨 라인 정리 → 승모 과개입 최소화 → 당기기 패턴 정렬 → 상체 후면 집중 강화
⸻
# 🏋️‍♀️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 언더그립 이지바 쉬러그 (후인)
- 목적 중·하부 승모 활성 + 견갑 후인 고정
- 포인트
	- 어깨를 위로 올리기 ❌
	- 견갑을 뒤로 모아 아래로 고정
	- 상단 1초 정지
	- 팔이 아닌 견갑 움직임 인지
- 🔑 체감 키워드
	- 등 위 중앙이 묵직하게 조여짐
	- 목 긴장 없음
	- 어깨 위치가 낮고 안정됨
⸻
2️⃣ 스미스 머신 언더그립 벤트오버 로우
- 목적 광배 하부·중부 등 두께 강화
- 포인트
	- 힙 힐지 자세 고정
	- 당기기 전 견갑 후인
	- 팔꿈치를 몸통 뒤로 길게
	- 반동 ❌
- 🔑 체감 키워드
	- 등 중앙이 깊게 조여짐
	- 광배 하부 사용감 선명
	- 허리 개입 없음
⸻
3️⃣ 로터리 풀다운 머신
- 목적 광배 하강 패턴 + 수직 당기기 안정
- 포인트
	- 견갑 하강 먼저
	- 팔로 당기지 말고 겨드랑이로 끌어내리기
	- 가슴 열림 유지
- 🔑 체감 키워드
	- 겨드랑이 아래부터 등으로 힘 전달
	- 어깨 들리지 않음
	- 내려갈수록 수축 명확
⸻
4️⃣ 덤벨 사이드 레터럴 레이즈
- 목적 측면 삼각근 라인 정리 + 어깨 분리 사용
- 포인트
	- 반동 ❌
	- 팔꿈치가 손보다 살쩍 위
	- 어깨 높이까지만
	- 상단 1초 정지
- 🔑 체감 키워드
	- 어깨 옵 라인이 타는 느낌
	- 승모 개입 최소
	- 좌우 흔들림 없음
⸻
# 📌 오늘 루틴 핵심 정리
- 견갑 안정이 모든 당기기의 시작
- 당기기 순서: → 하강 → 후인 → 팔
- 로우에서 허리 개입 느껴지면 무게 ↓
- 어깨 운동은 승모 힘 빼는 게 핵심
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 등 중앙 (중부 승모·능형근)
- 광배 하부
- 어깨 측면
✔ 운동 후 정상적인 느낌
- 어깨가 아래로 안정된 느낌
- 등이 두툘하게 사용된 느낌
- 목·승모 부담 없음
⚠ 체크 필요 신호
- 쉬러그 중 목 긴장
- 로우 동작에서 허리 뿌근함
- 풀다운 시 어깨 들림
- 레터럴 레이즈에서 반동 사용
