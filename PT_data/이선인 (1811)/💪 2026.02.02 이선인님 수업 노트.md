# 🧠 오늘 루틴 핵심 테마
가슴 상부 선활성 → 안정적인 프레스 패턴 확립 → 전면 어깨 분리 사용
→ 인클라인 각도에서 가슴 주도성 확보
→ 프레스 계열 동작에서 가슴 중심 밀기 패턴 강화
→ 전면 삼각근은 "들어 올리기"가 아닌 "밀어 올리기" 패턴 확립
→ 프론트 레이즈에서 덤벨 붙이는 힘으로 내전근 활성 극대화
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 인클라인 벤치 덤벨 체스트 플라이
- 목적 상부 가슴 이완·수축 인지 + 프레스 전 가슴 선활성
- 포인트
	- 견갑 고정, 팔꿈치 각도 유지
	- 내려갈수록 가슴이 열리는 범위까지만
	- 반동 ❌, 하단 컨트롤 필수
- 🔑 체감 키워드
	- 가슴 상부가 넓게 늘어나는 느낌
	- 어깨 개입 없음
	- 가슴 중앙 수축 선명
⸻
2️⃣ 체스트 프레스
- 목적 가슴 주도 안정적 밀기 패턴 확립
- 포인트
	- 시작 전 견갑 후인·하강 세팅
	- 팔로 밀기 ❌ → 가슴으로 받쳐 올리기
	- 궤적 일정, 상체 흔들림 ❌
- 🔑 체감 키워드
	- 가슴 중앙·상부가 동시에 차오름
	- 어깨 앞 힘 빠짐
	- 프레스 중 안정감 ↑
⸻
3️⃣ 프론트 숄더 프레스
- 목적 전면 삼각근 독립 사용 + 밀기 패턴 전환
- 포인트
	- 시작 위치에서 어깨 하강 고정
	- 승모 개입 ❌ → 순수 삼각근 밀기
	- 정수리 방향으로 수직 궤적 유지
- 🔑 체감 키워드
	- 어깨 앞쪽만 선명하게 켜짐
	- 목·승모 긴장 없음
	- 상단 수축 시 어깨 둥글게 형성
⸻
4️⃣ 덤벨 프론트 레이즈
(덤벨 두 개 붙여주는 힘)
- 목적 전면 삼각근 + 가슴 내전근 동시 활성
- 포인트
	- 덤벨을 서로 붙이는 힘 유지하며 들어 올리기
	- 팔꿈치 살짝 굽힌 상태 고정
	- 어깨 높이까지만 → 승모 개입 방지 ❌
	- 내려올 때도 덤벨 붙이는 힘 유지
- 🔑 체감 키워드
	- 가슴 안쪽 + 전면 어깨 동시 수축
	- 덤벨 붙이는 압력 느껴짐
	- 승모 긴장 없이 어깨만 작동
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 플라이로 가슴 선활성 → 프레스 계열로 밀기 패턴 확립
- 가슴 프레스는 "받쳐 올리기", 어깨 프레스는 "수직 밀기"
- 프론트 레이즈는 덤벨 붙이는 힘으로 내전근 활성 극대화
- 어깨 앞 긴장 느껴지면 → 무게 ↓ / 견갑 세팅 재확인
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 가슴 상부·중앙
- 전면 삼각근 (어깨 앞)
- 가슴 안쪽 (내전근 부위)
✔ 운동 후 정상적인 느낌
- 가슴이 단단하고 충만한 느낌
- 어깨가 아래로 안정된 사용감
- 팔 앞쪽 전반적인 펌핑감
⚠ 체크 필요 신호
- 프레스 후 어깨 앞 날카로운 통증
- 목·승모 과도한 긴장
- 프론트 레이즈 중 어깨 들림 느낌
- 가슴보다 팔에만 피로 집중
