# 🧠 오늘 루틴 핵심 테마
측면 삼각근 선활성 → 어깨 프레스 패턴 → 후면 어깨 활성 → 등 중앙 수축 → 광배 하강
→ 레터럴 레이즈로 측면 삼각근 분리 활성
→ 밀리터리 프레스로 어깨 전체 밀기 패턴 확립
→ 스미스 프레스로 안정적 어깨 밀기 강화
→ 업라이트 로우로 중부승모 + 측면 삼각근 연결
→ 페이스 풀로 후면 삼각근 분리 활성
→ 미드 로우로 등 중앙 수축 강화
→ 랫풀다운으로 광배 전체 활성
→ 벤트 오버 로우로 등 전체 마무리
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 덤벨 사이드 레터럴 레이즈
- 목적 측면 삼각근 분리 활성 + 어깨 선활성
- 포인트
	- 팔꿈치 살짝 굽힌 상태 유지
	- 어깨 높이까지만 들어 올리기
	- 승모 개입 ❌ → 순수 측면 삼각근
- 🔑 체감 키워드
	- 어깨 옆이 둥글게 켜지는 느낌
	- 승모 긴장 없음
	- 측면 삼각근만 선명
⸻
2️⃣ EZ-BAR 밀리터리 숄더 프레스
- 목적 어깨 전체 밀기 패턴 + 코어 안정성
- 포인트
	- 선 자세, 코어 단단히
	- 어깨 하강 고정 후 수직 밀기
	- 상체 반동 ❌
- 🔑 체감 키워드
	- 어깨 전체가 고르게 타는 느낌
	- 코어 안정감
	- 목·승모 힘 빠짐
⸻
3️⃣ 스미스 숄더 프레스
- 목적 어깨 안정적 밀기 + 프레스 깊이 확보
- 포인트
	- 시작 전 견갑 하강 세팅
	- 정수리 방향 수직 궤적
	- 어깨 앞 과개입 ❌
- 🔑 체감 키워드
	- 어깨 전체 균등하게 타격
	- 견갑 안정감
	- 상단 수축 선명
⸻
4️⃣ 케이블 업라이트 로우
- 목적 중부승모 + 측면 삼각근 동시 활성
- 포인트
	- 팔꿈치를 높이 들어 올리는 궤적
	- 팔꿈치 주도로 당기기
	- 어깨보다 팔꿈치가 높게
- 🔑 체감 키워드
	- 승모 중간 + 어깨 옆 동시 수축
	- 팔꿈치 높이 올라가는 감각
	- 어깨 앞 개입 최소
⸻
5️⃣ 케이블 로프 페이스 풀
- 목적 후면 삼각근 분리 활성 + 견갑 후인
- 포인트
	- 로프를 얼굴쪽으로 당기기
	- 견갑 후인 먼저, 팔꿈치 바깥
	- 어깨 들림 ❌
- 🔑 체감 키워드
	- 어깨 뒤쪽이 조이는 느낌
	- 견갑 후인 선명
	- 어깨 위치 안정적
⸻
6️⃣ 시티드 플레이트 로디드 미드 로우
- 목적 등 중앙 수축 + 견갑 후인 강화
- 포인트
	- 견갑 후인 먼저, 팔 당김
	- 상체 반동 ❌
	- 등 중앙으로 견갑 모으기
- 🔑 체감 키워드
	- 등 중앙 수축 선명
	- 견갑 후인 강력하게 느껴짐
	- 상체 안정적
⸻
7️⃣ 인클라인 벤치 케이블 랫풀다운
- 목적 광배 전체 활성 + 견갑 하강
- 포인트
	- 인클라인 벤치에 누워 견갑 하강 먼저
	- 가슴쪽으로 당기기
	- 팔꿈치 어깨 바로 아래
- 🔑 체감 키워드
	- 광배 전체 당겨지는 느낌
	- 견갑 단단히 고정
	- 가슴 수축감
⸻
8️⃣ 스미스 벤트 오버 로우
- 목적 등 전체 수축 + 광배·등 중앙 마무리
- 포인트
	- 상체 숯인 자세, 코어 고정
	- 견갑 후인 먼저, 팔 당김
	- 허리 굳히지 않게 ❌
- 🔑 체감 키워드
	- 등 전체 수축
	- 광배·등 중앙 동시 타격
	- 상체 안정적
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 레터럴 레이즈로 측면 삼각근 먼저 각성
- 프레스는 견갑 하강 → 수직 밀기
- 업라이트 로우는 팔꿈치 높이 = 궤적이 핵심
- 모든 로우는 견갑 후인 먼저 → 팔 당김
- 어깨 들리거나 승모 개입되면 → 견갑 세팅 재확인 / 무게 ↓
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 어깨 전체 (전면·측면·후면)
- 등 중앙
- 광배 중앙·하부
✔ 운동 후 정상적인 느낌
- 어깨가 둥글고 단단한 느낌
- 등과 어깨가 동시에 타는 느낌
- 견갑이 안정된 사용감
⚠ 체크 필요 신호
- 프레스 후 어깨 앞 통증
- 로우 동작에서 어깨 들림
- 업라이트 로우 중 승모 과긴장
- 벤트 오버 로우에서 허리 굳히는 느낌
