# 🧠 오늘 루틴 핵심 테마
평형성 + 햄스트링 각성 → 둔근 깊이 + 중둔근 → 정렬 패턴 → 대둔근 하부 마무리
→ 싱글레그 데드리프트로 평형성 + 햄스트링 깊이
→ 크로스 런지로 둔근 깊이 + 중둔근 동시 사용
→ 레그프레스로 발바닥-발목-무릎-골반 정렬 확립
→ 아웃타이로 대둔근 하부 고립 강화
# 🏋️‍♂️ 운동 루틴 + 🔑 동작별 체감 키워드
1️⃣ 케틀벨 싱글레그 스티프 데드리프트
- 목적 평형성 확립 + 햄스트링 깊이 활성 + 둔근 양측 균형
- 포인트
	- 한쪽 다리로 서기, 무릎 각도 고정
	- 골반 중립, 고관절을 뒤로 접는 움직임
	- 하단에서 햄스트링 깊이 느낀 후 엉덩이로 올라오기
	- 허리 굳히면 즉시 중단 ❌
- 🔑 체감 키워드
	- 햄스트링 단면이 깊게 늘어나는 느낌
	- 엉덩이 하부 긴장 형성
	- 평형 유지 감각
⸻
2️⃣ 스미스 크로스 런지
- 목적 둔근 깊이 활성 + 중둔근 동시 사용 + 골반 안정성
- 포인트
	- 뒤쪽 무릎을 깊게 굽혀 내려가기
	- 앞무릎 90도, 골반 흔들림 ❌
	- 일어서면서 엉덩이로 밀기
- 🔑 체감 키워드
	- 둔근이 깊이 늘어나다 수축되는 느낌
	- 엉덩이 옆 + 엉덩이 하부 동시 사용
	- 무릎 흔들림 없이 안정적
⸻
3️⃣ 레그프레스
(밴드 발목 ; 발바닥 - 발목-무릎-골반 정렬)
- 목적 하체 정렬 패턴 확립 + 하체 전체 활성
- 포인트
	- 발목에 밴드 걸기 (외전 유지 목적)
	- 발바닥 중앙 → 발목 → 무릎 → 골반 수직 정렬 확인
	- 발목이 안쪽으로 모이지 않게 ❌
	- 엉덩이로 밀어 올리기
- 🔑 체감 키워드
	- 하체 전체가 정렬된 느낌
	- 발바닥부터 힘이 전달되는 감각
	- 골반 안정적
⸻
4️⃣ 아웃타이 (대둔근 하부 타겟)
- 목적 대둔근 하부 고립 강화 + 둔근 형태 발달
- 포인트
	- 옆으로 누운 자세, 골반 중립
	- 무릎을 뒤쪽 대각선 방향으로 들어 올리기
	- 상단 1초 정지 후 컨트롤 하강
- 🔑 체감 키워드
	- 엉덩이 아래쪽의 단단한 느낌
	- 대둔근 하부만 선명하게 타격
	- 허벅지에 힘 안 들어감
⸻
# 📌 오늘 루틴 핵심 정리
⸻
📌 오늘 루틴 핵심 정리
- 싱글레그 데드리프트로 평형성 + 햄스트링 깊이 각성
- 크로스 런지는 "깊이 굽힌 후 엉덩이로 밀기"
- 레그프레스는 발바닥-발목-무릎-골반 수직 정렬이 핵심
- 아웃타이는 뒤쪽 대각선 = 대둔근 하부 타겟
- 골반 흔들림이나 허리 굳히면 → 포지션 재확인 / 무게 ↓
⸻
# ⭐ 예상 근육통 부위 안내
✔ 정상적으로 예상되는 근육통
- 햄스트링 전체
- 둔근 전체 (특히 하부)
- 엉덩이 옆 (중둔근)
✔ 운동 후 정상적인 느낌
- 보행·계단 시 엉덩이 사용감 ↑
- 하체 전체가 정렬된 느낌
- 엉덩이 아래쪽이 특히 단단함
⚠ 체크 필요 신호
- 데드리프트 중 허리 통증
- 런지 시 무릎 흔들림
- 레그프레스에서 발목 안쪽 모임
- 둔근보다 허벅지 앞 과도한 피로
